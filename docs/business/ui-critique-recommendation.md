# User Interface (UI) Critique & Recommendation Template

## 1. Executive Summary
- Purpose and context of the UI critique
- Key findings and recommendations

## 2. Evaluation Criteria
- Usability and accessibility
- Responsiveness and performance
- Visual design and consistency
- User feedback and satisfaction

## 3. Current UI Assessment
### 3.1 Strengths
- Positive aspects of the UI/UX

### 3.2 Weaknesses / Limitations
- Usability issues, accessibility gaps, or design inconsistencies
- Evidence/examples from user testing or feedback

### 3.3 Gaps vs. Requirements
- Comparison to UI/UX requirements/specifications
- Unmet needs or misalignments

## 4. Alternative Approaches Considered
- Other UI frameworks, design systems, or workflows
- Pros and cons of each

## 5. Recommendations
- Actionable improvements for UI/UX
- Prioritization and impact analysis

## 6. Implementation Guidance
- Steps for UI/UX enhancement
- Roles and responsibilities

## 7. Appendices
- User feedback, wireframes, references
