# Security Critique & Recommendation Template

## 1. Executive Summary
- Purpose and context of the security critique
- Key findings and recommendations

## 2. Evaluation Criteria
- Data security (encryption, access controls)
- Application security (authentication, authorization)
- Compliance (GDPR, auditability)
- Incident response readiness

## 3. Current Security Assessment
### 3.1 Strengths
- Security best practices in place

### 3.2 Weaknesses / Limitations
- Vulnerabilities, risks, or compliance gaps
- Evidence/examples from audits or incidents

### 3.3 Gaps vs. Requirements
- Comparison to security requirements/specifications
- Unmet needs or misalignments

## 4. Alternative Approaches Considered
- Other security frameworks, tools, or controls
- Pros and cons of each

## 5. Recommendations
- Actionable improvements for security posture
- Prioritization and impact analysis

## 6. Implementation Guidance
- Steps for remediation or enhancement
- Roles and responsibilities

## 7. Appendices
- Security audit reports, checklists, references
